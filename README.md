# Regulatory Affairs Egypt

A professional website for a regulatory affairs company specializing in product registration in Egypt.

[![Live Site](https://img.shields.io/badge/Live%20Site-View%20Here-blue)](https://[username].github.io/regulatory-affairs-egypt)

## Customization

Before publishing, replace the following placeholder values in `docs/index.html`:

- `[Company Name]` - Replace with your actual company name
- `[Your Phone Number]` - Replace with your phone number
- `[Your Email Address]` - Replace with your email address
- `[Your WhatsApp Number]` - Replace with your WhatsApp number

## How to Enable GitHub Pages

1. Go to your repository on GitHub
2. Click on **Settings** tab
3. Scroll down to **Pages** section
4. Under **Source**, select **Deploy from a branch**
5. Select **main** branch and **/docs** folder
6. Click **Save**

Your site will be available at `https://[username].github.io/regulatory-affairs-egypt`

## Local Preview

To preview the site locally:

1. Open `docs/index.html` in your web browser
2. The site will load with all styles and functionality
